# 网易云音乐缓存解码 NetEase Music Cache Decoder

*[English version of this reference](https://github.com/lchloride/NeteaseMusicCacheDecode/blob/master/readme_en.md)*

## 简介

将网易云音乐的uc!缓存文件转换成mp3音乐文件。提供多种运行程序以满足不同环境的需求。
